最終更新日: 2026年4月9日

## 第1条 (目的)
本規約は、Yuki Hamada（以下「当方」）が提供するmacOSアプリケーション「Trio」（以下「本アプリ」）の利用条件を定めるものです。

## 第2条 (利用登録)
本アプリの使用を開始した時点で本規約に同意したものとみなします。

## 第3条 (ライセンス)
当方はユーザーに対し、本規約の範囲内で本アプリを使用する非独占的・譲渡不能・取消可能なライセンスを付与します。以下の行為は禁止します:
- リバースエンジニアリング・逆コンパイル
- 本アプリのコピー配布・転売
- ライセンス表示の削除

## 第4条 (ユーザーの責任)
ユーザーは以下について単独で責任を負います:
1. 本アプリを合法的な目的でのみ使用すること
2. メッセージ処理対象アプリ（LINE、Slack等）の利用規約を遵守すること
3. 第三者のメッセージをAI処理する際、必要な同意を得ること
4. 業務利用時、所属組織のポリシーに従うこと
5. 自身の Anthropic API キー・アカウントの管理

## 第5条 (禁止事項)
以下の行為を禁止します:
- 他人になりすましてメッセージ送信
- ストーキング・ハラスメント目的の使用
- 他人のアカウントでの使用
- スパム送信
- 違法行為への使用
- 本アプリのセキュリティ機構を迂回する試み
- 本アプリを介した他サービスのAPI悪用
- 本アプリのサービス提供を妨害する行為

## 第6条 (第三者サービス依存)
本アプリは以下の第三者サービスに依存しており、これらのサービスが停止・変更された場合、本アプリの機能に影響が及ぶ可能性があります:
- Anthropic Claude API
- macOS システムAPI
- 各メッセンジャーアプリ
当方はこれらに起因する機能停止について責任を負いません。

### 7.1 保証の否認
本アプリは「現状のまま」提供され、以下について**いかなる保証もいたしません**:

- AI生成返信の内容の正確性・適切性
- アプリの中断なき動作
- 特定目的への適合性
- 第三者アプリとの互換性維持

### 7.2 責任制限
当方は以下について責任を負いません（故意・重過失による場合を除く）:

## LINE機能についての特記事項

本アプリの「LINE取得 (OCR)」機能は、**ユーザー自身のMac上で起動しているLINEアプリのウィンドウ**をVision frameworkでOCR処理する機能です。LINE株式会社の公式APIや内部データベースにはアクセスしません。

### 合法的な利用方法
- ✅ あなた自身のLINEアカウントの受信メッセージを、あなた自身のMac上でOCR
- ✅ 個人的な返信効率化

### 問題となりうる利用方法
- ⚠️ 他人のLINEアカウントを盗み見る目的
- ⚠️ 業務利用で相手方の同意なくAI処理する
- ⚠️ 取得した内容を第三者に販売・公開する

本機能を使用される際は、LINE株式会社の利用規約とユーザー自身の道徳的責任に従ってご判断ください。本機能に起因する LINE 株式会社または第三者との紛争については、当方は一切の責任を負いません。

問題があった場合、ユーザーの責任で本機能を無効化 (設定 > メッセンジャー > LINE トグルOFF) してください。
