# Trio v0.2 — 次の3機能

## 1. コマンドバーからLINE送信 (30分)
- `/send 粟田 メッセージ内容` でLINE送信
- `/send @slack #channel メッセージ` でSlack送信
- 検索バーが `/send` で始まったら送信モード

## 2. 人ベースUI (1時間)  
- メッセージごと→送信者ごとにグループ化
- 各人カード: アバター + 名前 + 最新メッセージ + 未読数
- クリック→その人のスレッド展開
- アイコン: LINEはOCRでプロフィール画像キャプチャ

## 3. Claude Code連携 (1時間)
- `/claude プロンプト` でclaude CLI実行
- 結果をUIに表示
- 結果を相手に送信するボタン
